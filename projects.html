PROJECTS

RuleMaster: Answers your burning board game rule questions using a Rule Answering Guru (RAG).
  • Next.js, Prisma, AWS S3, Postgres, pg_vector.
  • Parses and indexes boardgame rulebook PDFs, augmenting the context of ChatGPT3.5 with related text.
  • Created chat interface for asking questions.


SplitGPT: Resolve balances between friends using natural language processing and a debt simplification algorithm (2024).
  • Frontend: TypeScript, SvelteKit, Tailwind, Vercel.
  • Backend: Python, FastAPI, Heroku.
  • Uses ChatGPT3.5 and LangChain to extract structured data from text and decompose the problem into smaller tasks.
  • Experimented with prompts, agents vs. sequential chains, and more freedom vs. constrained outputs. Tried giving an agent math tools to do calculations, but it would make unnecessary tool calls and ultimately fail at the task. Currently using a simple sequential chain of prompts to fill in a table to the best of the chain's ability (without math tools), and letting the human edit the table on the frontend.
